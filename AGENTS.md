# AGENTS.md

## Creating new proposals

When creating a new proposal, use the template in the root of the repository:
`0000-template.md`.

Copy `0000-template.md` to `proposals/0000-your-feature.md`. Keep the `0000-`
prefix; it will be updated to the PR number once the pull request is created.
Fill out the proposal with as much detail as possible, following the structure
defined in the template.

See `README.md` for the full proposal submission process.
